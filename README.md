### Welcome to Liferay's Headless Team 👋

Pull requests for code affecting headless infrastructure / data integration / staging components are welcome to [liferay-headless/liferay-portal](https://github.com/liferay-headless/liferay-portal). See [PULL_REQUESTS.md](PULL_REQUESTS.md) for our PR guidelines and review workflow.

### Claude Code Skills

Install:

```bash
curl -sSL https://raw.githubusercontent.com/liferay-headless/liferay-headless/main/scripts/skills_install.sh | sh
```

Uninstall:

```bash
curl -sSL https://raw.githubusercontent.com/liferay-headless/liferay-headless/main/scripts/skills_uninstall.sh | sh
```

### liferay.com launch script

Run from the parent directory of your `liferay-portal` checkout (with `liferay.com.zip` next to it):

```bash
curl -sSL https://raw.githubusercontent.com/liferay-headless/liferay-headless/main/scripts/liferay.com.sh | bash
```
